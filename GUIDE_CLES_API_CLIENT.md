# 🎯 Ce guide vous permet de créer vos comptes en 15 minutes

Vous allez créer 2 comptes gratuits :
1. **Stripe** → pour recevoir les paiements par carte bancaire

# PARTIE 1 : CRÉER VOTRE COMPTE STRIPE (Paiements)

## Étape 1 : Aller sur le site

1. Ouvrez votre navigateur internet (Google Chrome, Firefox, Safari...)
2. Dans la barre d'adresse en haut, tapez : **stripe.com**
3. Appuyez sur Entrée

---

## Étape 2 : Créer un compte

1. Cliquez sur le bouton bleu **"Commencer"** ou **"Start now"**
2. Remplissez :
   - **Email** : votre adresse email professionnelle
   - **Nom complet** : Nicolas Bernard
   - **Mot de passe** : choisissez un mot de passe sécurisé (notez-le !)
3. Cliquez sur **"Créer un compte"**

---

## Étape 3 : Vérifier votre email

1. Allez dans votre boîte email
2. Ouvrez l'email de Stripe
3. Cliquez sur le lien **"Vérifier mon email"**

---

## Étape 4 : Configurer votre compte (informations légales)

Stripe vous demandera :
- **Type d'activité** : Auto-entrepreneur / Particulier
- **Pays** : France
- **Votre nom** : Nicolas Bernard
- **Date de naissance**
- **Adresse postale**
- **Numéro SIRET** (si vous en avez un)
- **IBAN** : pour recevoir vos paiements

➡️ Remplissez toutes les informations demandées honnêtement.

---

## Étape 5 : Récupérer vos clés API (IMPORTANT)

1. Une fois connecté, regardez en haut à gauche : cliquez sur **"Développeurs"**
2. Dans le menu, cliquez sur **"Clés API"**
3. Vous verrez 2 clés :
   
   ```
   Clé publique : pk_live_xxxxxxxxxxxxx
   Clé secrète  : sk_live_xxxxxxxxxxxxx
   ```

4. **Cliquez sur "Révéler la clé secrète"** pour la voir
5. **COPIEZ CES 2 CLÉS** et envoyez-les moi par email ou SMS

⚠️ **Ne partagez jamais ces clés publiquement !**

## Étape 6 : Créer le Webhook (je vous aiderai)

📞 **Appelez-moi ou envoyez-moi un message quand vous arrivez à cette étape.**

Je vous guiderai pour cette partie ou je le ferai avec vous en partage d'écran.

# PARTIE 2 : CRÉER VOTRE COMPTE SENDGRID (Emails)

## Étape 1 : Aller sur le site

1. Dans votre navigateur, tapez : **sendgrid.com**
2. Appuyez sur Entrée

---

## Étape 2 : Créer un compte gratuit

1. Cliquez sur **"Start for Free"** (Commencer gratuitement)
2. Remplissez :
   - **Email** : votre adresse email
   - **Mot de passe** : choisissez un mot de passe (notez-le !)
3. Cliquez sur **"Create Account"**

---

## Étape 3 : Vérifier votre email

1. Allez dans votre boîte email
2. Ouvrez l'email de SendGrid
3. Cliquez sur le lien de vérification

---

## Étape 4 : Remplir le questionnaire

SendGrid pose quelques questions :
- **What is your website?** → votre futur site (vous pouvez mettre : nicolas-bernard-therapie.fr)
- **How many emails do you plan to send?** → Less than 10,000 (moins de 10 000)
- **What will you be sending?** → Transactional emails (emails de transaction)

---

## Étape 5 : Vérifier votre identité d'expéditeur

1. Dans le menu à gauche, cliquez sur **"Settings"** (Paramètres)
2. Puis cliquez sur **"Sender Authentication"**
3. Cliquez sur **"Verify a Single Sender"**
4. Remplissez :
   - **From Email** : votre email professionnel (ex: contact@nicolas-bernard.fr)
   - **From Name** : Nicolas Bernard
   - **Adresse**, etc.
5. Cliquez sur **"Create"**
6. Allez dans votre boîte email et cliquez sur le lien de vérification

---

## Étape 6 : Créer votre clé API (IMPORTANT)

1. Dans le menu à gauche, cliquez sur **"Settings"**
2. Puis cliquez sur **"API Keys"**
3. Cliquez sur le bouton bleu **"Create API Key"**
4. **API Key Name** : tapez `site-therapie`
5. **Permissions** : sélectionnez **"Full Access"**
6. Cliquez sur **"Create & View"**
7. **COPIEZ LA CLÉ AFFICHÉE** (elle commence par `SG.`)

⚠️ **Cette clé ne s'affiche qu'une seule fois ! Copiez-la immédiatement !**

Envoyez-moi cette clé par email ou SMS.
